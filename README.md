# Sistema Web PetShop - Fase 1

Este repositório contém os arquivos referentes à Fase 1 do Projeto da Disciplina de desenvolvimento web.

## Descrição do Projeto
A Fase 1 consiste na construção estrutural de uma página web para um Petshop utilizando exclusivamente HTML, sem o uso de customizações (CSS e Bootstrap) ou implementações de funções em JavaScript. O foco desta etapa é a correta aplicação das tags semânticas do HTML5 para organizar a informação de maneira lógica, preparando o sistema para a Fase 2.

## Funcionalidades da Página
* **Cabeçalho (Header):** Contém o nome do petshop, uma breve descrição e um menu de navegação interno com âncoras para rolagem rápida até as seções de Produtos, Serviços e Contato.
* **Catálogo de Produtos (Body):** Exibição estruturada das mercadorias divididas nas 3 categorias exigidas:
  * Acessórios (roupas, brinquedos, camas, etc.)
  * Rações não perecíveis
  * Produtos de higiene e limpeza (tapete higiênico, fraldas, etc.)
  * *Nota: Cada categoria lista 2 produtos acompanhados de sua respectiva imagem, descrição detalhada e valor.*
* **Catálogo de Serviços (Body):** Lista os serviços disponíveis no petshop. Informa a descrição de cada serviço, o valor e a disponibilidade da modalidade "tele-busca".
* **Rodapé (Footer):** Exibe endereço, telefone, autoria do projeto e links de navegação para retornar às seções superiores da página.
